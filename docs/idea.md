# AIRadar — 아이디어

> Vibe UX · brainstorming-idea 산출물
> 작성일: 2026-06-12

## 한 줄 컨셉
AI 분야에 깊이 파고드는 사람을 위해, 여러 소스에서 핫한 콘텐츠를 매일 자동 수집·요약해 웹에 게시하는 AI 큐레이션 서비스

## 문제 정의
매일 아침 AI 분야 정보를 얻으려면 유튜브, aitimes 등 여러 사이트를 직접 돌아다녀야 한다. 전문가들이 어디서 정보를 얻는지 모르기 때문에 좋은 소스 자체를 놓치는 경우도 많다. 특정 주제를 깊게 팔 때 참고할 수 있는 핫한 콘텐츠를 한 곳에서 바로 볼 수 없어 불편하다.

## 타깃 사용자
AI 분야를 꾸준히 팔로우하는 개인 — 특정 주제(LLM, 로보틱스, AI 툴 등)를 깊게 파고들 때 최신 핫 콘텐츠를 효율적으로 수집하고 싶은 사람. PC나 아이패드에서 여유 있게 탐색하는 스타일.

## 핵심 솔루션 (메인 컨셉)
유튜브, 뉴스 사이트(aitimes 등), 커뮤니티(Reddit, Hacker News 등) 같은 여러 소스에서 조회수·좋아요 기준으로 AI 분야 핫 콘텐츠를 자동 수집한다. AI가 각 콘텐츠를 한두 줄로 요약하고, 세부 주제별로 분류해 매일 웹 페이지에 게시한다. 처음엔 개인 큐레이션 도구로 쓰다가, 완성도가 높아지면 비슷한 관심사를 가진 사람들과 공유하는 공개 플랫폼으로 확장한다.

## 핵심 기능 목록
- 다중 소스 자동 수집 — 유튜브, 뉴스 사이트, 커뮤니티 등에서 AI 콘텐츠 자동 크롤링
- 인기도 기반 랭킹 — 조회수 1순위, 좋아요 2순위로 상위 5~10개 선별
- AI 자동 요약 — 각 콘텐츠를 한두 줄로 요약해 빠르게 파악 가능
- 주제별 분류 및 필터링 — AI 세부 분야(LLM, 로보틱스, AI 툴 등)로 분류해 원하는 주제만 탐색
- 매일 자동 게시 — 큐레이션 결과를 웹 페이지에 일별 포스트 형태로 게시

## 확장 아이디어
- 공개 플랫폼 전환 후 구독·팔로우 기능
- 특정 주제 알림 설정
- 독자가 직접 소스 추가 제안

## 플랫폼
- 형태: 반응형 웹 (기본)
- 기준 뷰포트: 데스크톱 우선 (PC·아이패드 환경 기준, 정보량 많은 화면을 넓게 활용)
