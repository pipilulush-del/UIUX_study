## 플로우 1 · 첫 방문·일별 탐색 — "오늘 AI 뭐 핫해?"  [메인 기능: 인기도 랭킹 + AI 요약 + 저장하기]

**목표 순간:** 아침에 접속 → 랭킹 카드를 훑다가 요약 읽고 "이거 봐야겠다" 하고 저장하거나 바로 클릭하는 장면  
(상태 변화: 아무것도 모름 → 오늘 핫한 AI 콘텐츠 파악 + 관심 것 저장)

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 (→ 전진) | 보조 |
|:--:|------|------|------------------|------|
| ① | 홈 | 진입·가치 파악 | 피드로 시선 이동 → ② | — |
| ② | 홈 — 랭킹 피드 | 인기도 랭킹 + AI 요약 + 소스 태그 | 카드 훑기 → ③ | — |
| ③ | 홈 — 카드 | AI 요약 읽기 + 저장하기 | 🔖 탭 → ② 계속 탐색 / 제목 클릭 → ④ | 저장 토스트 |
| ④ | 원문 (외부) | — | 새 탭으로 이동 (플로우 종료) | — |

**화면**

### ① 홈 (진입)

```
┌──────────────────────────────────────────────────────────────┐
  AIRadar                                       [ 내 저장 목록 ]
  ─────────────────────────────────────────────────────────────
  YouTube · 뉴스 · 커뮤니티 AI 핫 콘텐츠 매일 자동 수집

  📅 2026년 6월 12일 (목)  ·  오늘의 AI 랭킹

  [ 전체 ]  [ LLM ]  [ 로보틱스 ]  [ AI 툴 ]  [ 연구·논문 ]

  ┌────────────────────────────────────────────────────────┐
  │  1  제목이 여기 들어갑니다                        [🔖] │
  │     ▶ YouTube · #LLM · 조회수 1.2M                    │
  │     AI 요약 한두 줄. 핵심 내용 빠르게.                │
  └────────────────────────────────────────────────────────┘
  ┌────────────────────────────────────────────────────────┐
  │  2  두 번째 콘텐츠 제목                           [🔖] │
  │     📰 뉴스 · #로보틱스 · 조회수 850K                 │
  │     요약...                                            │
  └────────────────────────────────────────────────────────┘
                        ...
└──────────────────────────────────────────────────────────────┘
```

- **역할:** 서비스 한 줄 설명 + 오늘 날짜 + 토픽 필터 탭 + 랭킹 피드 진입
- **담는 정보·요소:** 헤더(로고·한 줄 설명·저장 목록 링크) · 날짜·섹션 타이틀 · 토픽 필터 탭 · 랭킹 카드 목록
- **주 행동:** 카드 훑기 시작 → ②

### ② 홈 — 랭킹 피드 (카드 상세)

```
┌────────────────────────────────────────────────────────────┐
│  1  ┌──────┐  제목이 여기 들어갑니다               [🔖]   │
│     │ 썸네 │  ▶ YouTube · #LLM · 조회수 1.2M              │
│     │ 일   │  AI 요약 한두 줄. 핵심 내용 빠르게.          │
│     └──────┘                                               │
├────────────────────────────────────────────────────────────┤
│  2  제목이 여기 들어갑니다                          [🔖]   │
│     📰 뉴스 · #로보틱스 · 조회수 850K                     │
│     AI 요약 한두 줄.                                       │
└────────────────────────────────────────────────────────────┘
```

- **역할:** YouTube 카드는 썸네일 노출, 뉴스·커뮤니티는 텍스트만 — 소스 타입별 자연스러운 구분
- **담는 정보·요소:** 랭킹 번호 · 썸네일(YouTube만) · 제목(원문 링크) · 소스 타입 아이콘 · 토픽 태그 · 조회수 · AI 요약 · 🔖
- **주 행동:** 🔖 탭 → ③ / 제목 클릭 → ④

### ③ 저장 완료 (토스트)

```
  ┌────────────────────────────────┐
  │  저장됐어요  내 목록 보기 →   │  ← 하단 토스트 (2초 후 자동 사라짐)
  └────────────────────────────────┘
```

- **역할:** 저장 확인 + 내 저장 목록 화면 존재를 첫 저장 시 안내
- **주 행동:** 무시 → ② 계속 탐색 / "내 목록 보기" 탭 → 플로우 3

### ④ 원문 (외부)

- 새 탭으로 열림. 플로우 종료.

**핵심 가치:** 랭킹 카드 한 화면에서 AI 요약을 읽고 관심 콘텐츠를 저장까지 — 탭 이동 없이 오늘의 AI 핫 콘텐츠 파악 완료

## 플로우 2 · 주제·소스 필터 탐색 — "LLM만 모아보고 싶다"  [메인 기능: 주제별 분류·필터링]

**목표 순간:** 토픽 탭 하나 탭 → LLM 관련 핫 콘텐츠만 바로 보임  
(상태 변화: 전체 피드(노이즈) → 관심 토픽 집중 탐색)

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 (→ 전진) | 보조 |
|:--:|------|------|------------------|------|
| ① | 홈 — 토픽·소스 필터 | 주제별 분류·필터링 | 탭 선택 → ② | 소스 타입 보조 필터 |
| ② | 홈 — 필터된 피드 | 인기도 랭킹 + AI 요약 | 요약 읽기 + 저장 / 원문 이동 | 필터 해제(×) |

**화면**

### ① 홈 — 토픽·소스 필터

```
  [ 전체 ]  [ LLM ]  [ 로보틱스 ]  [ AI 툴 ]  [ 연구·논문 ]   ← 토픽 탭

  [ 전체 소스 ]  [ ▶ YouTube ]  [ 📰 뉴스 ]  [ 💬 커뮤니티 ]  ← 소스 타입 탭
```

- **역할:** 토픽과 소스 타입을 독립적으로 선택 — 조합 필터 가능 (예: LLM × YouTube)
- **주 행동:** 탭 선택 → 피드 인플레이스 갱신 → ②

### ② 홈 — 필터된 피드

```
┌────────────────────────────────────────────────────────────┐
│  [ LLM ]  ×                [ ▶ YouTube ]  ×               │
│  ─────────────────────────────────────────────────────     │
│  1  ┌──────┐  제목                                [🔖]    │
│     │ 썸네 │  ▶ YouTube · #LLM · 조회수 1.2M             │
│     │ 일   │  AI 요약 한두 줄.                            │
│     └──────┘                                               │
├────────────────────────────────────────────────────────────┤
│  2  ┌──────┐  제목                                [🔖]    │
│     └──────┘  ▶ YouTube · #LLM · 조회수 740K             │
│               AI 요약...                                   │
└────────────────────────────────────────────────────────────┘
```

- **역할:** 활성 필터 표시(× 로 해제 가능) + 필터 조합에 맞는 랭킹 피드
- **담는 정보·요소:** 활성 필터 칩(해제 버튼 포함) · 필터된 랭킹 카드 목록
- **주 행동:** 저장·원문 이동 (플로우 1 ③④와 동일)

**핵심 가치:** 토픽 + 소스 타입 조합 필터 한 번으로 관심 분야만 바로 집중 탐색 — YouTube 포함이라는 차별점이 소스 필터로 명시적으로 드러남

## 플로우 3 · 내 저장 목록 — "저장해둔 거 다시 볼까"  [메인 기능: 저장 목록 조회]

**목표 순간:** 헤더의 "내 저장 목록" 탭 → 아껴둔 콘텐츠 한 번에 확인  
(상태 변화: 흩어진 관심 콘텐츠 → 한 곳에 모아 재탐색)

**단계 시퀀스**

| 단계 | 화면 | 기능 | 주 행동 (→ 전진) | 보조 |
|:--:|------|------|------------------|------|
| ① | 내 저장 목록 | 저장 목록 조회 | 카드 훑기 → ② | 빈 상태 안내 |
| ② | 내 저장 목록 — 카드 | 저장 목록 조회 | 원문 이동 / 🔖✓ 탭 → 저장 해제 | — |

**화면**

### ① 내 저장 목록 — 빈 상태

```
┌──────────────────────────────────────────────────────────────┐
  AIRadar                                       [ 내 저장 목록 ]
  ─────────────────────────────────────────────────────────────
  내 저장 목록

      🔖
      아직 저장된 콘텐츠가 없어요.
      관심 있는 콘텐츠의 🔖 를 탭하면 여기 모여요.

                      [ 오늘 랭킹 보러 가기 ]

└──────────────────────────────────────────────────────────────┘
```

### ① 내 저장 목록 — 목록 있는 상태

```
┌──────────────────────────────────────────────────────────────┐
  AIRadar                                       [ 내 저장 목록 ]
  ─────────────────────────────────────────────────────────────
  내 저장 목록  · 3개

  ┌──────────────────────────────────────────────────────────┐
  │  ┌──────┐  제목                                  [🔖✓]  │
  │  │ 썸네 │  ▶ YouTube · #LLM · 2026.06.12                │
  │  └──────┘  AI 요약 한두 줄.                             │
  └──────────────────────────────────────────────────────────┘
  ┌──────────────────────────────────────────────────────────┐
  │  제목                                           [🔖✓]   │
  │  📰 뉴스 · #AI 툴 · 2026.06.11                         │
  │  AI 요약 한두 줄.                                        │
  └──────────────────────────────────────────────────────────┘
└──────────────────────────────────────────────────────────────┘
```

- **역할:** 로컬 저장된 콘텐츠 목록 (저장 날짜 최신순) · 해제는 🔖✓ 다시 탭 (토글)
- **담는 정보·요소:** 저장 수 · 카드(썸네일·제목·소스·토픽·저장일·요약) · 저장 해제 토글
- **주 행동:** 제목 클릭 → 원문 이동 / 🔖✓ 탭 → 저장 해제

**핵심 가치:** 별도 계정 없이 로컬에서 바로 — 관심 콘텐츠를 언제든 꺼내볼 수 있는 개인 아카이브

**참고:** 로컬 스토리지 기반이라 다른 기기에서 목록이 공유되지 않음. 공개 플랫폼 전환 시 계정 동기화로 확장 예정.
